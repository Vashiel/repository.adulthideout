Install In Kodi

  1. Open Kodi and go to Settings, then File manager.
  2. Select Add source and enter https://vashiel.github.io/repository.adulthideout/.
  3. Name the source AdultHideout.
  4. Open Add-ons, choose Install from zip file, then select repository.adulthideout-1.0.4.zip.
  5. Use Install from repository to install or update the AdultHideout video addon.

About This Repository

AdultHideout is an adult-oriented Kodi addon project for users who want a maintained video addon repository with direct update metadata. The repository package currently installs the AdultHideout repository, and the addon manifest publishes AdultHideout version 1.0.12 "Signal Lock".

This page is the canonical public homepage for the repository source. It is intended for adults only and does not replace the Kodi installation workflow.
